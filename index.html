Fetch JSON data from Datawrapper

This tool lets you fetch the underlying JSON data from Datawrapper charts. This is something you might find handy if you create charts using the public API of Datawrapper and need to understand exactly what properties are available for each chart type.

Get it here: https://app.datawrapper.de/account/api-tokens
